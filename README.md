# Yiwen (Jacqueline) Chen — Portfolio

Personal portfolio site for Yiwen (Jacqueline) Chen — arts & entertainment marketer and producer.

Static HTML/CSS/JS. Entry point: `index.html`. All images, video, and the resume PDF live in `assets/`.

## Run locally
Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8099
```

Then visit http://localhost:8099
